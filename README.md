# Navigation
Using MapNav for navigation tasks
Here will try to understand RL better by using few small examples. This program is inspired by course given by Mutual information. Here we will do rover Bellman Equation, Dynamic Programming, Generalized Policy Iteration, Monte Carlo, Off-Policy methods, Temporal Difference learning, Function Approximation to finally, Policy gradient methods.
Here is a overview about the story along the introduction process. Before starting with Bellman equation. We first make assumption about MDP, MDP predicts future using the current information. The information set consists of S, A, R, T, discount(meaning state, action, reward, trajectory and discount for reward with respect to time). MDP makes assumption that future decision and rewards depend on current information, not the past. Since for general case we should have continuous states, we will start with countable states and we will approximate general continuous states using function approximation methods in the future.
